
## Reflexión

¿Qué fue lo más difícil?
Lo más difícil fue entender cómo conectar correctamente los archivos entre sí y subir el proyecto a GitHub.

¿Qué no entendí completamente?
No entendí del todo cómo funcionan los eventos en JavaScript ni todos los comandos de Git.

¿Para qué sirve Git en proyectos reales?
Git sirve para tener en cuenta los cambios realizados, trabajar en equipo y poder volver a versiones anteriores si algo falla.

¿Por qué es importante estructurar bien un proyecto?
Es importante porque hace que el código sea más ordenado y fácil de entender.
